# CS253 Software DevOps Assignment 1

## Project Overview
This project is a complete implementation of the software development and DevOps practices learned in the CS253 course. It aims to demonstrate the best practices in software development lifecycle, including version control, continuous integration/deployment, testing, and documentation.

## Project Structure
The project is structured as follows:

```
CS253-Software-DevOps-Assignment1/
├── src/                   # Source code
│   └── main/             # Main application logic
│   └── tests/            # Test scripts
├── docs/                 # Documentation files
├── .gitignore            # Git ignore file
├── README.md             # Project documentation
└── requirements.txt      # Python dependencies
```

## Usage Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/krishna-kumar-bais/CS253-Software-DevOps-Assignment1.git
   cd CS253-Software-DevOps-Assignment1
   ```
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the application**:
   ```bash
   python src/main/app.py
   ```
4. **Run tests**:
   ```bash
   pytest src/tests/
   ```
